# /clint:configure

Pergunte ao usuário, **uma por vez**, as credenciais necessárias pro MCP `clint`. As variáveis esperadas geralmente estão no README do projeto em https://github.com/thaleslaray/clint-mcp.

Salve em `~/.claude/plugins/data/clint/config.json` (criar diretórios se necessário).

Avise o user que precisa **reiniciar o Claude Code** pra o MCP carregar as creds.
