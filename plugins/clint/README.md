# clint

MCP server for the Clint CRM API (Brazilian) — 46 endpoints, FastMCP + Code Mode

## Instalação (Claude Code)

```
/plugin marketplace add thaleslaray/plugins
/plugin install clint
/clint:configure
```

## Instalação (Claude Desktop)

Baixa o `.mcpb` na [release mais recente](https://github.com/thaleslaray/clint-mcp/releases/latest) e dá duplo-clique.
