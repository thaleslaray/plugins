# /hotmart:configure

Pergunte ao usuário, **uma por vez**, as credenciais necessárias pro MCP `hotmart`. As variáveis esperadas geralmente estão no README do projeto em https://github.com/thaleslaray/hotmart-mcp.

Salve em `~/.claude/plugins/data/hotmart/config.json` (criar diretórios se necessário).

Avise o user que precisa **reiniciar o Claude Code** pra o MCP carregar as creds.
